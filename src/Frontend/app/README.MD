# src/Frontend/app

This directory is intended for the React and Vite frontend application.

The following files are expected to live here eventually:
- `package.json` - npm dependencies and scripts.
- `vite.config.ts` - Vite configuration.
- `orval.config.ts` - Orval configuration for generating hooks and the API client.

Planned frontend stack:
- React.
- TanStack Router.
- TanStack Query.
- Orval for client generation based on the backend contract.
