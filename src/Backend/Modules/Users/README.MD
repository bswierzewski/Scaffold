# src/Backend/Modules/Users

This directory is intended for the `Scaffold.Modules.Users` module.

It is expected to contain:
- `Scaffold.Modules.Users.csproj` - a `classlib` project.
- endpoints related to users.
- module business services.
- module models and contracts.
- an optional `DbContext` and persistence configuration for the Users area.

The module should remain independent from the Weather module and other domain modules.
