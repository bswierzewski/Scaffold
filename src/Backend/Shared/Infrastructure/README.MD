# src/Backend/Shared/Infrastructure

This directory is intended for the `Scaffold.Shared.Infrastructure` project.

It is expected to contain:
- `Scaffold.Shared.Infrastructure.csproj` - a `classlib` project.
- C# extensions, for example `AddCustomAuth`.
- shared EF Core configuration.
- integration components used by multiple modules or by the host.
- shared dependency registration and application infrastructure configuration.
