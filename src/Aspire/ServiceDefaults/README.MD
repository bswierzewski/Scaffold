# src/Aspire/ServiceDefaults

This directory is intended for the `Scaffold.ServiceDefaults` project.

It is expected to contain:
- `Scaffold.ServiceDefaults.csproj` - a `classlib` project.
- shared OpenTelemetry configuration.
- HealthChecks.
- default logging configuration, for example Serilog or another shared logging mechanism.
- extensions consumed by other services in the solution.
